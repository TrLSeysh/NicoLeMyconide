# NicoLeMyconide
```PYTHON 3.10``` ```PYGAME 2.5.2```

---

*N'avez-vous jamais rêvé de vous occuper d'une petite pousse de myconide ? C'est désormais possible ! 
Occupez-vous de Nico et prennez soin de lui sinon il sera très triste :(*

Ce projet est réalisé dans le cadre des cours de M1 Python Avancé du Domaine Professionnel IA de l'ISEN Brest.

---

## Les fonctionnalités

Occupez-vous de Nico avec plusieurs activités :

#### **Biberon | Le nourrir :** 
>Un carton contenant de la nourriture apparaîtra à proximité de Nico, touchez cette boite libèrera la nourriture, recliquez dessus et vous lui donnerez à manger !
#### **Energie | Le mettre au lit :** 
>Cette adorable créature a également besoin de sommeil. En cliquant sur le bouton cela le fera dormir.
#### **Savon | Le laver :** 
>Nico doit rester propre ! Cliquez sur le bouton pour le laver. (la fonctionnalité est à améliorer)
#### **Coeur | Le carresser :** 
>Ce petit champignon a besoin de tendresse. Cliquer sur le bouton changera votre curseur pour vous permettre de le carresser (attention à ne pas le faire quand il dort !). Pour arrêter cliquez sur la croix en bas à droite.

*PS : La fonction pour charger une partie est encore en cours d'implémentation.*

## Lancer le jeu

> Le jeu utilise la library pygame. Les autres libraries sont toutes inclues dans python par défaut.

#### Pour installer les libraries :

```pip install -r requirements.txt```

#### Pour lancer le jeu :
*(les commandes sont à executer dans le dossier Nico)*

- sous linux : ```python3 main.py```
- sous windows : ```python main.py```
